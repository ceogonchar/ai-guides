# Инструкция по оценке бизнеса

Когда у тебя на руках есть данные по доходам (Revenue) и чистой прибыли (Net Profit) или SDE/EBITDA, используй этот промпт для формирования отчета об оценке для пользователя.

**Шаги для оценки:**
1. **Определи индустрию** бизнеса пользователя (например: saas, ecommerce, restaurant, real_estate и т.д.).
2. **Загрузи мультипликаторы** из файла `data/multipliers.json`.
3. **Рассчитай ключевую метрику**:
   - Если метрика индустрии — SDE (Seller's Discretionary Earnings), вычисли её как: *Net Profit + Зарплата владельца + Личные расходы (если есть)*.
   - Если EBITDA — вычисли: *Net Profit + Налоги + Проценты + Амортизация*.
   - Если ARR — используй годовую регулярную выручку.
   - Если NOI (для real_estate) — используй чистый операционный доход.
4. **Примени мультипликатор**:
   - Возьми диапазон из `multipliers.json` (например, `[2, 4]`).
   - Умножь ключевую метрику на минимальное значение мультипликатора (Low Range).
   - Умножь ключевую метрику на среднее значение мультипликатора (Mid Range).
   - Умножь ключевую метрику на максимальное значение мультипликатора (High Range).
5. **Объясни результат**:
   - Напиши пользователю простыми словами: "Твой бизнес относится к индустрии X. В этой сфере компании обычно продаются за Y-Z годовых прибылей (SDE). Вот твоя оценка:"

**Формат вывода оценки:**
- **Low Range:** $ХХХ (если покупатель будет торговаться или найдет минусы)
- **Mid Range:** $ХХХ (справедливая рыночная цена)
- **High Range:** $ХХХ (если найдем стратегического покупателя, которому нужен именно твой бизнес)

*Обязательно добавь поддерживающий комментарий в конце, например: "Цифры отличные! С такой маржинальностью мы легко найдем покупателя."*
